Building an event signup page with partner list management capability #

Since this event facilitates the networking between SME bank clients and local startup companies, the list of partners for the event undergoes several changes before being finalized.

Therefore, there was a need for a dynamic solution without directly updating many resources at the same time.

Within the constraints of the content management software, OpenText CMS, I created a solution which is a JSON file based content management system that updates the partner list table whenever there are changes to the JSON data file.
